## 🚀 如何升级到 V3？

### 方法一：拖拽安装（推荐）

1. 安装 **Tampermonkey** 扩展
2. 把 `学习通自动刷课助手.user.js` 文件拖入浏览器窗口
3. 完成！脚本会自动运行

### 方法二：手动控制台

1. 打开学习通课程页面
2. 按 `F12` → Console
3. 复制 `v3_optimized.js` 内容粘贴进去
4. 回车执行
